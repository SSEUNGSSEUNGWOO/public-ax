# PUBLIC-AI · Civic Bracket
Logo concept 01 — Civic Bracket (공적 프레임)

## Files
- symbol.svg                 64x64, ink + orange accent
- symbol-mono.svg            64x64, single-color (ink)
- symbol-on-dark.svg         64x64, for dark backgrounds (paper + orange)
- lockup-horizontal.svg      420x120, symbol + wordmark
- lockup-horizontal-on-dark.svg
- lockup-vertical.svg        240x200, symbol over wordmark
- lockup-vertical-on-dark.svg
- favicon.svg                32x32, rounded ink tile

## Tokens
- Primary  oklch(0.68 0.18 50)  ≈ #f5912e
- Ink      #0A0A0B
- Paper    #fbfaf7

## Wordmark rule
Space Grotesk 600, letter-spacing -0.02em.
"-" hyphen at 40% opacity, "AI" in primary orange.

## Notes
The bracket [ ] is the mark — a frame for civic information.
Three center dots = the signal, last dot orange = the transition signal.
For application in dark UI, use the on-dark variants.
